# 業務時間記録

ローカルPC上で動作する、個人用の業務時間記録アプリです。  
サーバー、ログイン、外部ライブラリは不要です。

## 使い方

1. `index.html` をブラウザで開きます。
2. 業務カテゴリのボタンを押すと記録が開始されます。
3. 作業を中断するときは「停止」を押します。
4. 作業を再開するときは「再開」を押します。
5. 作業が終わったら「終了」を押します。

同時に作業中にできる記録は1件だけです。  
別のカテゴリを押すと、現在の記録は自動で終了し、新しいカテゴリの記録が開始されます。

## 業務カテゴリ

- 会議
- 打合せ
- 議会対応
- 窓口対応
- 電話対応
- 資料作成
- 作業
- 訪問
- 現場
- その他

「その他」を押した場合は、カテゴリ名を入力してから開始します。

## 履歴の編集と削除

「全履歴一覧」の各行にある「編集」から、以下の項目を修正できます。

- カテゴリ
- その他カテゴリ名
- 開始日時
- 終了日時
- 状態
- 実作業時間
- 停止時間

「削除」を押すと、その1件だけを削除できます。削除前には確認メッセージが表示されます。

## CSV出力

「CSV出力」を押すと、ブラウザに保存されている全ログをCSVファイルとして出力します。  
CSVはUTF-8 BOM付きです。

出力列は以下です。

```text
log_id,work_date,work_category,work_category_custom,started_at,ended_at,work_minutes,pause_minutes,status
```

## 保存場所

記録データはブラウザの `localStorage` に保存されます。  
キーは `workTimeLogs` です。

ブラウザのデータ削除や「全データ削除」を行うと記録は消えます。必要に応じてCSV出力でバックアップしてください。

## キーボードショートカット

| キー | 操作 |
|---|---|
| 1 | 会議を開始 |
| 2 | 打合せを開始 |
| 3 | 議会対応を開始 |
| 4 | 窓口対応を開始 |
| 5 | 電話対応を開始 |
| 6 | 資料作成を開始 |
| 7 | 作業を開始 |
| 8 | 訪問を開始 |
| 9 | 現場を開始 |
| 0 | その他を開始 |
| S | 停止 |
| R | 再開 |
| E | 終了 |

入力欄やセレクトボックスにフォーカスがある場合、ショートカットは無効です。
