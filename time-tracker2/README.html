業務時間記録 使い方

ローカルPCで使う個人用の業務時間記録アプリです。

アプリへ戻る

基本操作

  1. 業務カテゴリのボタンを押すと、その業務の記録が開始されます。
  2. 作業を中断するときは「停止」を押します。
  3. 作業を再開するときは「再開」を押します。
  4. 作業が終わったら「終了」を押します。

同時に作業中にできる記録は1件だけです。別のカテゴリを押すと、現在の記録は自動で終了し、新しいカテゴリの記録が開始されます。

その他カテゴリ

「その他」を押すとカテゴリ名の入力欄が表示されます。入力後に「開始」を押すか、Enterキーで開始できます。

履歴の編集と削除

「全履歴一覧」の各行にある「編集」から、カテゴリ、開始日時、終了日時、状態、実作業時間、停止時間を修正できます。

「削除」を押すと、その1件だけを削除できます。削除前に確認メッセージが表示されます。

CSV出力

「CSV出力」を押すと、ブラウザに保存されている全ログをCSVファイルとして出力します。CSVはUTF-8 BOM付きなので、Excelでも文字化けしにくい形式です。

保存場所

記録データはブラウザの localStorage に保存されます。サーバーやログインは使いません。

ブラウザのデータ削除や「全データ削除」を行うと記録は消えます。必要に応じてCSV出力でバックアップしてください。
