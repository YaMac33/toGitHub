# 業務時間可視化

`time-tracker` から出力したCSVを読み込み、業務カテゴリごとの時間を集計・可視化するローカル用アプリです。  
サーバー、ログイン、外部ライブラリは不要です。

## 使い方

1. `index.html` をブラウザで開きます。
2. 「CSVファイルを選択」を押します。
3. `time-tracker` から出力したCSVファイルを選択します。
4. 読み込み後、グラフ・集計表・ログ一覧が自動で更新されます。

複数CSVをまとめて読み込めます。上限は100ファイルです。

## 対応CSV

`time-tracker` のCSV出力で作成された、以下の列を持つCSVに対応しています。

```text
log_id,work_date,work_category,work_category_custom,started_at,ended_at,work_minutes,pause_minutes,status
```

UTF-8 BOM付きCSV、カンマ、ダブルクォート、改行を含むCSV値にも対応しています。

## 表示される集計

- 総実作業時間
- 総停止時間
- 1日平均
- 最多カテゴリ
- 日別の実作業時間
- カテゴリ別の実作業時間
- 曜日別の実作業時間
- 状態別の件数
- カテゴリ集計表
- 読み込みログ一覧

## 絞り込み

以下の条件で表示対象を絞り込めます。

- 開始日
- 終了日
- カテゴリ
- 状態

絞り込みを変更すると、グラフ、集計表、ログ一覧が同時に更新されます。

## カテゴリ集計表

カテゴリごとに以下を表示します。

- 件数
- 実作業時間
- 停止時間
- 1件平均
- 構成比

構成比は、絞り込み後の総実作業時間に対して、そのカテゴリが占める割合です。

## 読み込みログ一覧

読み込んだログを一覧表示します。  
最新100件を対象に表示します。

## 注意点

このアプリはCSVを読み込んで表示するだけです。  
`time-tracker` の `localStorage` データは変更しません。

画面を再読み込みすると、読み込んだCSVの状態はリセットされます。必要な場合は再度CSVを選択してください。
