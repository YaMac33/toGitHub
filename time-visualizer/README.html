業務時間可視化 使い方

time-tracker から出力したCSVを集計・可視化する別アプリです。

アプリへ戻る

概要

このアプリは、time-tracker の「CSV出力」で作成したCSVファイルを読み込み、業務カテゴリごとの時間を確認するためのローカル用ツールです。

サーバーやログインは使いません。読み込んだCSVの内容は画面上で集計されます。

CSVの読み込み

  1. 「CSVファイルを選択」を押します。
  2. time-tracker から出力したCSVファイルを選びます。
  3. 複数ファイルをまとめて選択できます。
  4. 読み込み後、各グラフと集計表が自動で更新されます。

一度に読み込めるCSVファイルは最大100個です。

表示される集計

  • 総実作業時間
  • 総停止時間
  • 1日平均
  • 最多カテゴリ
  • 日別の実作業時間
  • カテゴリ別の実作業時間
  • 曜日別の実作業時間
  • 状態別の件数
  • カテゴリ集計表
  • 読み込みログ一覧

絞り込み

開始日、終了日、カテゴリ、状態で表示対象を絞り込めます。絞り込みを変更すると、グラフ・集計表・ログ一覧も同時に更新されます。

カテゴリ集計表

カテゴリごとに、件数、実作業時間、停止時間、1件平均、構成比を表示します。

構成比は、絞り込み後の総実作業時間に対して、そのカテゴリが占める割合です。

注意点

このアプリはCSVを読み込んで表示するだけで、time-tracker の localStorage データは変更しません。

画面を再読み込みすると、読み込んだCSVの状態はリセットされます。必要な場合は再度CSVを選択してください。
