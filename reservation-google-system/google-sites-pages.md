# Googleサイトのページ構成案と貼り付け文章

## ページ構成

- トップページ
- 予約状況
- 予約申請
- 利用ルール
- 管理者向け

## トップページ

### 貼り付け文章

```text
予約ポータル

このサイトでは、施設・場所の予約状況確認と予約申請ができます。

予約を希望する方は、まず「予約状況」ページで空き状況を確認し、「予約申請」ページから申請してください。
フォーム送信時点では予約は確定していません。
管理者が内容を確認し、承認メールの送信をもって予約確定となります。

利用前に「利用ルール」を必ず確認してください。
```

## 予約状況ページ

このページには、Apps ScriptでデプロイしたWebアプリURLを埋め込みます。Googleカレンダーの埋め込みではありません。

### 貼り付け文章

```text
予約状況

現在承認済みの予約を表示しています。
月表示、週表示、日表示、一覧表示を切り替えて確認できます。

同じ利用場所で時間帯が重なる申請は承認できない場合があります。
最新の状況を確認してから予約申請してください。
```

### 埋め込み

```text
挿入 → 埋め込む → URL
Apps Script WebアプリURLを貼り付け
```

## 予約申請ページ

このページにはGoogleフォームを埋め込みます。

### 貼り付け文章

```text
予約申請

以下のフォームから予約を申請してください。

フォーム送信後、受付メールが自動送信されます。
この時点では予約は確定していません。
管理者が申請内容と予約状況を確認し、承認メールの送信をもって予約確定となります。

申請内容に不備がある場合や、他の予約と重複する場合は、却下または調整をお願いすることがあります。
```

## 利用ルールページ

### 貼り付け文章

```text
利用ルール

1. 予約は承認メールの送信をもって確定します。
2. フォーム送信のみでは予約確定ではありません。
3. 利用時間には準備と片付けの時間を含めて申請してください。
4. 同じ利用場所で時間帯が重なる予約は承認できません。
5. 申請内容に変更がある場合は、管理者へ連絡してください。
6. 予約を取り消す場合は、できるだけ早めに管理者へ連絡してください。
7. 利用後は原状復帰をお願いします。
8. 施設や備品の破損、トラブルがあった場合は速やかに管理者へ連絡してください。
```

## 管理者向けページ

このページは、公開範囲を管理者のみにすることを推奨します。

### 貼り付け文章

```text
管理者向け

予約申請は、回答先スプレッドシートの「申請一覧」で確認します。

運用手順:

1. 新規申請のステータスは自動で「未処理」になります。
2. 内容を確認し、承認する場合はステータスを「承認」に変更します。
3. 重複がなければGoogleカレンダーに予定が登録され、申請者へ承認メールが送信されます。
4. 承認しない場合はステータスを「却下」に変更します。
5. 承認済み予約を取り消す場合はステータスを「取消」に変更します。
6. 処理結果や重複チェック結果は、スプレッドシートの各列で確認できます。

注意:
ステータス列以外を編集しても自動処理は実行されません。
カレンダーイベントID、処理日時、処理結果、重複チェック結果は原則として手入力しないでください。
```
