# 回答先スプレッドシートの列設計

シート名は `申請一覧` にします。

## フォーム回答列

Googleフォームから自動で作成される列です。

| 列 | 列名 | 内容 |
|---|---|---|
| A | タイムスタンプ | フォーム送信日時 |
| B | メールアドレス | 申請者のメールアドレス |
| C | 氏名 | 申請者名 |
| D | 所属 | 申請者の所属 |
| E | 利用場所 | 予約対象の場所・施設 |
| F | 希望日 | 利用日 |
| G | 開始時刻 | 利用開始時刻 |
| H | 終了時刻 | 利用終了時刻 |
| I | 利用目的 | 利用目的 |
| J | 備考 | 申請者の補足 |

## 管理用に追加する列

フォーム回答列の右側に追加します。

| 推奨列 | 列名 | 内容 |
|---|---|---|
| K | ステータス | `未処理`、`承認`、`却下`、`取消` |
| L | 管理メモ | 管理者用のメモ |
| M | カレンダーイベントID | GASが作成したGoogleカレンダーイベントID |
| N | 処理日時 | GASが最後に処理した日時 |
| O | 処理結果 | 登録成功、通知済み、エラーなど |
| P | 重複チェック結果 | 重複なし、または重複内容 |

## ステータス値

| 値 | 意味 |
|---|---|
| 未処理 | 申請受付済み、管理者確認待ち |
| 承認 | 重複チェック後、カレンダー登録と承認メール送信 |
| 却下 | 却下メール送信 |
| 取消 | 登録済みカレンダー予定の削除または取消表示、取消通知送信 |

## 保護推奨

- フォーム回答列は原則編集しない運用にします。
- `ステータス`、`管理メモ` は管理者だけが編集できるようにします。
- `カレンダーイベントID`、`処理日時`、`処理結果`、`重複チェック結果` はGASが更新する列のため、通常は手入力しません。
