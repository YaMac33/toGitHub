説明書

窓口受付当番表作成アプリの使い方と仕様まとめです。

アプリに戻る

概要

このアプリは、1か月分の窓口受付当番表をブラウザだけで作成するためのツールです。 外部ライブラリやCDNは使わず、HTML、CSS、JavaScriptのみで動作します。

土日と手動登録した祝日は割当対象から除外し、平日のCW 3枠、CW以外 4枠へ職員を自動割当します。

基本的な使い方

  1. 「サンプル職員を読み込む」または職員一覧から職員を追加します。
  2. 必要に応じて、休暇日や祝日を登録します。
  3. 年と月を選択します。
  4. 「自動作成」を押すと、対象月の当番表が作成されます。
  5. 必要に応じて、セル入れ替えや名前の書き換えで微調整します。
  6. 完成後、「CSV出力」からCSVファイルを保存できます。

自動割当の主なルール

  • 1日につきCW 午前・CW 昼・CW 午後・CW以外 午前・CW以外 昼1・CW以外 昼2・CW以外 午後の7枠を埋めます。
  • CW枠にはCW、CW以外枠にはCW以外の職員だけを割り当てます。
  • 同じ日に同じ職員を複数枠へ割り当てません。
  • activeがOFFの職員は割当対象外です。
  • 休暇登録されている職員は、その日の候補から除外します。
  • 同じ職員が同じ枠で連続しないようにします。
  • 同じ職員が同じ曜日・同じ枠を月内で重複担当しないようにします。
  • 対象になる枠それぞれの担当回数が、職員ごとにできるだけ均等になるようにします。
  • 同じ人の担当が月曜日または金曜日に偏りすぎないようにします。

月別ルール

月別ルールでは、選択中の月だけに適用される目標回数の調整を登録できます。 たとえば「前月少なかった人を今月は+1回」などの調整に使います。

現在は職員ごとの目標回数を+1、+2、-1、-2で調整できます。 今後ルールを増やす場合も、月別ルールのデータとして拡張しやすい構成です。

手動調整

セル入れ替え

当番表内の名前を2つクリックして選択し、「選択した2枠を入れ替え」を押すと、 その2枠だけを入れ替えます。

名前の書き換え

当番表内の名前を1つクリックし、「選択中の1枠目をこの職員に変更」から職員を選んで 「名前を書き換え」を押すと、そのセルだけ担当者を変更できます。

入れ替えや書き換えの際も、休暇、同日重複、同じ枠の連続、同じ曜日・同じ枠の重複などの 必須ルールを検証します。違反する場合は変更されません。

保存とCSV

  • 職員、休暇、祝日、月別ルール、作成済み当番表はlocalStorageに保存されます。
  • ブラウザを再読み込みしても、前回の状態を復元できます。
  • CSVはBOM付きUTF-8で出力します。
