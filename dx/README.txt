 DX推進リーダー 活動紹介ページ  更新手順書（README）
====================================================================

このサイトは、共有フォルダ内のHTMLファイルをそのまま開いて見る
「オフライン・完全ローカル」のサイトです。
インターネットにはつながっていません。

このフォルダを丸ごと共有フォルダに置き、index.html を開けば表示されます。
（ブラウザは Microsoft Edge を想定しています）


--------------------------------------------------------------------
 フォルダの中身
--------------------------------------------------------------------

DX推進リーダー活動紹介/
├── index.html          … トップページ（記事一覧）。基本さわりません。
├── style.css           … 見た目の設定。さわりません。
├── layout.js           … 記事の一覧を管理するファイル。★ここに1行追記します
├── pages/              … 記事のHTMLを入れる場所
├── info/               … 固定ページ（about / contact）。必要時のみ編集
├── images/             … 画像を入れる場所（ここに1箇所集約）
├── _template/          … 新しい記事を作るときの雛形
└── README.txt          … この手順書


--------------------------------------------------------------------
 新しい記事を追加する手順（全体で3ステップ）
--------------------------------------------------------------------

■ ステップ1：雛形をコピーして記事を作る

 1. _template フォルダの「記事テンプレート.html」をコピーする
 2. コピーしたファイルを pages フォルダに移動する
 3. ファイル名を、次のルールで付け直す

    ルール：  日付_内容.html   （すべて半角）
    例    ：  2026-08-01_meeting-room.html

    ・日付は  YYYY-MM-DD  の形（2026-08-01 など）
    ・内容は  半角の英数字とハイフン  だけ（日本語やスペースは使わない）


■ ステップ2：記事の中身を書く

 4. pages に置いたファイルを右クリック →「プログラムから開く」
    →「メモ帳」で開く
 5. ファイルの中の【書き換え1〜4】と書かれた場所だけを書き換える
    （それ以外の場所はさわらない）
 6. 上書き保存する


■ ステップ3：一覧に載せる（★ここが一番大事）

 7. layout.js を メモ帳で開く
 8. 上のほうにある「記事リスト（ARTICLES）」を探す
 9. 次の3行のかたまりを1つコピーして、すぐ上に貼り付ける

      {
        file: "2026-07-02_dx-results.html",
        title: "DX推進リーダー これまでの取り組みと実績",
        date: "2026-07-02",
        category: ""
      },

 10. 貼り付けたほうの中身を、今回の記事に書き換える
      file  … ステップ1で付けたファイル名（正確に）
      title … 一覧に表示したいタイトル
      date  … 日付（YYYY-MM-DD）
      category … 空っぽ（"" のまま）でOK

 11. 上書き保存する

 ※ 注意：各かたまりの最後（閉じカッコ } の後ろ）に
    カンマ「,」が必要です。付け忘れると一覧が表示されなくなります。
    いちばん最後の記事だけはカンマ不要です（元からあるお手本を参考に）。

 12. index.html を開いて、一覧に新しい記事が出ていれば完成です。


--------------------------------------------------------------------
 画像を入れたいとき
--------------------------------------------------------------------

 1. 画像ファイルを images フォルダに入れる
 2. ファイル名は次のルールで付ける（すべて半角）

    日付_記事名_連番.jpg
    例：2026-08-01_meeting-room_01.jpg

 3. 記事HTMLの中の【任意：画像を入れる場合】の説明にしたがって、
    画像の行を有効にし、ファイル名を書き換える


--------------------------------------------------------------------
 うまく表示されないとき
--------------------------------------------------------------------

・一覧に記事が出ない
   → layout.js のカンマ「,」の付け忘れ・付けすぎがないか確認

・見た目が崩れる（文字だけになる）
   → style.css / layout.js が index.html と同じ場所にあるか確認
   → 記事は pages フォルダ、固定ページは info フォルダに入っているか確認
     （場所が変わると「../」の参照がずれて読み込めません）

・画像が出ない
   → images フォルダに画像があるか、ファイル名が一致しているか確認
     （大文字小文字・全角半角も区別されます）


--------------------------------------------------------------------
 さわってはいけないファイル
--------------------------------------------------------------------

 ・style.css       （見た目全体の設定）
 ・layout.js の「記事リスト」以外の部分

 これらは全ページ共通です。壊すと全ページに影響します。
 変更が必要なときは、DX推進担当に相談してください。
