# ダウンロード案内

ご購入ありがとうございます。

この記事の有料部分では、「PDF書き込みビューアー」の配布ZIPを利用するための案内をまとめています。

ZIPファイルをダウンロードしたら、まず任意の場所に解凍してください。
解凍後、フォルダ内の index.html をブラウザで開くとツールを起動できます。

# ファイル構成

配布ZIPの中身は、以下の構成です。

/pdf-annotator/
  index.html
  style.css
  app.js
  README.txt
  third-party-licenses.txt
  /lib/
    /pdfjs/
      pdf.min.js
      pdf.worker.min.js
    /pdf-lib/
      pdf-lib.min.js

index.html が起動用ファイルです。
style.css は見た目の設定、app.js はアプリの動作をまとめたファイルです。

lib フォルダには、PDF表示用の PDF.js と、PDF保存用の pdf-lib が入っています。

# 使い方

基本的な使い方は以下の通りです。

1. zipを解凍する
2. pdf-annotator フォルダを開く
3. index.html をブラウザで開く
4. 「PDF選択」からPDFを選ぶ
5. 必要な注釈を書き込む
6. 「保存」から編集済みPDFをダウンロードする

ブラウザは Chrome を推奨しています。

# 操作方法

## PDF選択

画面上部の「PDF選択」から、編集したいPDFを選びます。
PDFを選択するまでは、ページ移動や保存など一部の操作はできません。

## 描画

「描画」を選ぶと、PDF上に線を書き込めます。
自由線と直線を選択できます。

また、種類から「ペン」または「マーカー」を選べます。
マーカーは半透明の線として表示されます。

## テキスト

「テキスト」を選んでPDF上をクリックすると、テキスト入力欄が表示されます。
入力後、「確定」を押すと注釈として配置されます。

配置済みのテキストをクリックすると、内容の修正や削除ができます。

## 消しゴム

「消しゴム」を選ぶと、注釈を消せます。

消し方は以下から選べます。

・オブジェクト
・通った部分

「オブジェクト」は、選択した注釈単位で削除します。
「通った部分」は、なぞった部分だけを消します。

## 元に戻す

直前の注釈操作を戻します。

## やり直し

元に戻した操作をやり直します。

## 拡大

PDFの表示倍率を上げます。

## 縮小

PDFの表示倍率を下げます。

## 現在ページをクリア

現在表示しているページの注釈を削除します。
他のページの注釈は残ります。

## 保存

注釈を書き込んだPDFを保存します。
元のPDF本文を直接編集するのではなく、注釈をPDFに焼き込んだ新しいPDFとしてダウンロードします。

# よくあるつまずき

## 先にPDFを選択する必要があります

PDFを選択していない状態では、描画や保存などの操作はできません。
まず「PDF選択」からPDFを開いてください。

## 大容量PDFは重くなる場合があります

ページ数が多いPDFや画像が多いPDFでは、表示や保存に時間がかかる場合があります。

## ブラウザはChrome推奨です

基本的にはPCブラウザでの利用を想定しています。
特に Chrome での利用を推奨します。

## スマートフォンでは操作しにくい場合があります

スマートフォンでも開ける場合がありますが、画面サイズやタッチ操作の都合で操作しにくいことがあります。
PCブラウザでの利用がおすすめです。

# カスタマイズ例

このツールは HTML / CSS / JavaScript で作られています。
必要に応じて、以下のような軽いカスタマイズができます。

## style.cssで見た目を変更できる

ボタンの色、背景色、余白、文字サイズなどは style.css で調整できます。

## app.jsで初期色や初期倍率を変更できる

初期のペン色、線の太さ、表示倍率などは app.js 内の設定値を変更することで調整できます。

## index.htmlでボタン文言を変更できる

画面に表示されるボタン名やラベルは index.html で変更できます。

# 仕組み

このツールは、既存PDFの本文を直接編集するものではありません。

仕組みは以下の通りです。

・PDF.jsでPDFを表示
・canvasで注釈を描画
・pdf-libで注釈をPDFに焼き込む
・PDF本文を直接編集するのではなく、注釈を重ねて保存時に焼き込む

そのため、PDF本文の文字を書き換える用途には向いていません。
あくまで、PDFの上にメモや線、テキスト注釈を追加するためのツールです。

# 安全性

このツールは、PDFをブラウザ内で読み込み、注釈の描画や保存処理もブラウザ内で行います。

確認した範囲では、PDFを外部送信する処理や不審な外部通信処理は見当たりませんでした。

ただし、これはAIによる簡易レビューであり、安全性を完全に保証するものではありません。
利用するPDFの内容や利用環境に応じて、ご自身の判断でご利用ください。

# 利用上の注意

・大容量PDFでは動作が重くなる場合があります
・ブラウザは Chrome を推奨します
・PDF本文そのものの文字編集はできません
・保存時は、注釈を焼き込んだ新しいPDFとして出力します
・重要なPDFを扱う場合は、元ファイルのバックアップを残してから利用してください
