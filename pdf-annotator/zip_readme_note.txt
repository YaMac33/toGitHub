# このファイルについて

このZIPには、ブラウザだけで使える「PDF書き込みビューアー」が入っています。

インストールは不要です。
ZIPを解凍して index.html を開くと利用できます。

# 起動方法

1. ZIPを解凍します
2. pdf-annotator フォルダを開きます
3. index.html をブラウザで開きます
4. 「PDF選択」からPDFを選びます

ブラウザは Chrome を推奨します。

# 基本操作

・PDF選択：編集したいPDFを開きます
・描画：PDF上に線を書き込みます
・テキスト：PDF上にテキスト注釈を追加します
・消しゴム：注釈を削除します
・元に戻す：直前の操作を戻します
・やり直し：戻した操作をやり直します
・拡大：表示倍率を上げます
・縮小：表示倍率を下げます
・現在ページをクリア：表示中ページの注釈を削除します
・保存：注釈を書き込んだPDFをダウンロードします

# 注意事項

・先にPDFを選択してから操作してください
・大容量PDFは動作が重くなる場合があります
・スマートフォンでは操作しにくい場合があります
・PDF本文の文字を直接編集するツールではありません
・重要なPDFを扱う場合は、元ファイルのバックアップを残してから利用してください
